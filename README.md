# PACE Framework Docs

Official documentation for the [PACE Framework](https://github.com/pace-framework-org/pace-framework-starter) — built with [Astro Starlight](https://starlight.astro.build/).

Live at: **[pace-framework.org](https://pace-framework.org)**

## Development

```bash
npm install
npm run dev
```

## Contributing

Edit the markdown files in `src/content/docs/`. The sidebar is configured in `astro.config.mjs`. Submit a pull request — the site deploys automatically to GitHub Pages on merge to `main`.
